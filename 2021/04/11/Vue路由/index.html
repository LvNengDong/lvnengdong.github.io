Vue路由

Posted on 2021-04-11 In Vue

什么是路由

路由就是一组映射关系，即<K, V>键值对。

K 是 Path，

V 是 通过这个 Path 要访问的路径！

  • 在后端中，V 一般是处理请求的回调函数。
  • 在前端（Vue）中：V 就是一个组件。

Vue-Router 概述

  • Vue-Router 是官方提供的用来实现 SPA 的 vue 插件
  • GitHub地址：https://github.com/vuejs/vue-router
  • 官方文档（中文）：http://router.vuejs.org/zh-cn/
  • 下载：npm install vue-router --save

Vue-Router 相关 API

1、创建路由器

**VueRouter()**：用于创建路由器的构造函数

路由器 & 路由：路由器是路由信息的集合，路由器是一个构造函数，在路由器的构造函数中可以配置多个路有信息！

2、在路由器中配置路由信息

3、注册路由器

  • 注册路由器就是声明使用路由器，我们创建好了 Vue-Router 实例后如果不告诉 Vue 的话 Vue 在启动项目时也不会去加载Vue-Router这个路由器实例。声明使用类似于 SpringBoot 中的 @Component 注解，只有声明使用后项目才会去加载该类中的信息。

  • Vue 一般是一个单页面应用，只会绑定一个 Vue 对象（当然这个 Vue 对象可以有任意多个子级配置对象），我们需要在这个 Vue 实例中声明使用路由器，项目在加载时才会去加载路由器中配置的路由信息。

4、使用路由组件标签

  1. <router-link>：用来生成路由链接
  2. <router-view>：用来显示当前路由组件界面

快速入门（基本路由）

在实际开发中，组件可分为两大类：

  1. 路由组件：路由组件一般放在 views/pages 目录下
  2. 非路由组件：非路由组件一般放在 components 目录下

在实际开发中，一般会专门创建一个 src/router/index.js 文件用于定义路由器模块！
