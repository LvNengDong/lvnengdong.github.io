2 Redisдёӯзҡ„ж•°жҚ®зұ»еһӢ

RedisйҖҡиҝҮжҸҗдҫӣеӨҡз§Қй”®еҖјж•°жҚ®зұ»еһӢжқҘйҖӮеә”дёҚеҗҢеңәжҷҜдёӢзҡ„еӯҳеӮЁйңҖжұӮпјҢзӣ®еүҚдёәжӯўRedisж”ҜжҢҒзҡ„й”®еҖјж•°жҚ®зұ»еһӢеҰӮдёӢпјҡ

RedisеңЁеҶ…еӯҳдёӯд»Ҙй”®еҖјеҜ№зҡ„еҪўејҸеӯҳеӮЁж•°жҚ®гҖӮе…¶дёӯжүҖжңүй”®пјҲkeyпјүйғҪжҳҜStringзұ»еһӢпјҢиҖҢеҖјпјҲvalueпјүжңү5з§ҚдёҚеҗҢзҡ„ж•°жҚ®з»“жһ„пјҢжҲ‘д»¬еёёиҜҙзҡ„Redisзҡ„ж•°жҚ®зұ»еһӢе…¶е®һжҳҜжҢҮRedisй”®еҖјеҜ№дёӯеҖјпјҲvalueпјүзҡ„ж•°жҚ®зұ»еһӢгҖӮ

  1. еӯ—з¬ҰдёІзұ»еһӢ String
  2. е“ҲеёҢзұ»еһӢ hash пјҡ Mapж јејҸ
  3. еҲ—иЎЁзұ»еһӢ list пјҡ LinkedListж јејҸпјҢж”ҜжҢҒйҮҚеӨҚе…ғзҙ 
  4. йӣҶеҗҲзұ»еһӢ set пјҡ LinkedListж јејҸпјҢдёҚе…Ғи®ёйҮҚеӨҚе…ғзҙ 
  5. жңүеәҸйӣҶеҗҲзұ»еһӢ sortedset пјҡ LinkedListж јејҸпјҢдёҚе…Ғи®ёйҮҚеӨҚе…ғзҙ пјҢдё”е…ғзҙ жңүеәҸгҖӮ

3 Redisе‘Ҫд»Өж“ҚдҪң

3.1 еӯ—з¬ҰдёІзұ»еһӢпјҲStringпјү

  1. еӯҳеӮЁпјҡset key value

  2. иҺ·еҸ–пјҡget key

  3. еҲ йҷӨпјҡdel key

3.2 е“ҲеёҢзұ»еһӢпјҲHashпјү

  1. еӯҳеӮЁпјҡhset key field value

  2. иҺ·еҸ–пјҡ

    • иҺ·еҸ– Hash дёӯжҢҮе®ҡ key зҡ„еҖјпјҡhget key hashKey
    • иҺ·еҸ– Hash дёӯзҡ„жүҖжңүй”®еҖјеҜ№пјҡhgetall key

  3. еҲ йҷӨпјҡhdel key hashKey

3.3 еҲ—иЎЁзұ»еһӢпјҲListпјү

  1. ж·»еҠ пјҡеҸҜд»Ҙж·»еҠ дёҖдёӘе…ғзҙ еҲ°еҲ—иЎЁзҡ„еӨҙйғЁпјҲе·Ұиҫ№пјүжҲ–е°ҫйғЁпјҲеҸіиҫ№пјү

    • lpush key valueпјҡе°Ҷе…ғзҙ еҠ е…ҘеҲ—иЎЁе·Ұиҫ№
    • rpush key valueпјҡе°Ҷе…ғзҙ еҠ е…ҘеҲ—иЎЁеҸіиҫ№

    Listе’ҢStringзҡ„еҢәеҲ«жҳҜпјҡеҪ“keyзӣёеҗҢж—¶еӯҳеӮЁе…ғзҙ пјҢStringзұ»еһӢжҳҜиҰҶзӣ–пјҢиҖҢListжҳҜиҝҪеҠ 

  2. иҺ·еҸ–пјҡlrange key start endпјҢж №жҚ®иҢғеӣҙиҺ·еҸ–гҖӮ0 -1иЎЁзӨәиҺ·еҸ–е…ЁйғЁе…ғзҙ гҖӮ

    • дёӢж Үд»Һ 0 ејҖе§ӢеҸ–пјҢе·ҰеҸійғҪжҳҜй—ӯеҢәй—ҙпјҢд№ҹе°ұжҳҜиҜҙ lrange mylist 0 0 д№ҹиғҪеҸ–еҲ°з¬¬дёҖдёӘе…ғзҙ 
    • жІЎжңүrrangeиҝҷдёӘе‘Ҫд»Ө

  3. еҲ йҷӨпјҡ

    • lpop keyпјҡеҲ йҷӨеҲ—иЎЁжңҖе·Ұиҫ№зҡ„е…ғзҙ пјҢе№¶е°Ҷе…ғзҙ иҝ”еӣһ
    • rpop keyпјҡеҲ йҷӨеҲ—иЎЁжңҖеҸіиҫ№зҡ„е…ғзҙ пјҢе№¶е°Ҷе…ғзҙ иҝ”еӣһ

3.4 йӣҶеҗҲзұ»еһӢпјҲSetпјү

дёҚе…Ғи®ёйҮҚеӨҚе…ғзҙ 

  1. еӯҳеӮЁпјҡsadd key value

  2. иҺ·еҸ–йӣҶеҗҲдёӯзҡ„жүҖжңүе…ғзҙ пјҡsmembers key

  3. еҲ йҷӨйӣҶеҗҲдёӯзҡ„жҹҗдёӘе…ғзҙ пјҡsrem key value

3.5 жңүеәҸйӣҶеҗҲзұ»еһӢпјҲSortedSetпјү

дёҚе…Ғи®ёйҮҚеӨҚе…ғзҙ пјҢдё”е…ғзҙ жңүеәҸпјҢжҜҸдёӘе…ғзҙ йғҪдјҡе…іиҒ”дёҖдёӘdoubleзұ»еһӢзҡ„еҲҶж•°пјҲscoreпјүпјҢRedis йҖҡиҝҮеҲҶж•°пјҲscoreпјүжқҘдёәйӣҶеҗҲдёӯзҡ„жҲҗе‘ҳд»Һе°ҸеҲ°еӨ§жҺ’еәҸгҖӮ

  1. еӯҳеӮЁпјҡzadd key score value

  2. иҺ·еҸ–пјҡ

    • zrange key start endпјҡжҢүеҲҶж•°д»Һе°ҸеҲ°еӨ§зҡ„йЎәеәҸиҺ·еҸ–value
    • zrange key start end withscoresпјҡжҢүеҲҶж•°д»Һе°ҸеҲ°еӨ§зҡ„йЎәеәҸиҺ·еҸ–valueе’Ңscore

  3. еҲ йҷӨпјҡzrem key value

3.6 йҖҡз”Ёе‘Ҫд»Ө

  1. жҹҘиҜўжүҖжңүзҡ„й”®пјҡkeys *
  2. иҺ·еҸ–жҢҮе®ҡй”®keyеҜ№еә”зҡ„еҖјvalueзҡ„ж•°жҚ®зұ»еһӢпјҡtype key
  3. еҲ йҷӨжҢҮе®ҡзҡ„й”®еҖјеҜ№пјҡdel key

4 жҢҒд№…еҢ–

Redis жҳҜдёҖдёӘеҶ…еӯҳж•°жҚ®еә“пјҢеҪ“RedisжңҚеҠЎеҷЁйҮҚеҗҜпјҢжҲ–иҖ…жңҚеҠЎеҷЁз”өи„‘йҮҚеҗҜеҗҺж•°жҚ®е°ұдјҡдёўеӨұгҖӮдёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢжҲ‘д»¬еҸҜд»Ҙе°ҶRedisеҶ…еӯҳдёӯзҡ„ж•°жҚ®жҢҒд№…еҢ–еҲ°зЎ¬зӣҳзҡ„ж–Үд»¶дёӯгҖӮ

Redis дёӯзҡ„жҢҒд№…еҢ–жңәеҲ¶пјҲ2з§Қпјү

  2. AOF

RDB

RDBпјҡй»ҳи®Өж–№ејҸпјҢж— йңҖиҝӣиЎҢй…ҚзҪ®пјҢй»ҳи®Өе°ұдҪҝз”Ёиҝҷз§ҚжңәеҲ¶гҖӮ

иҝҷз§ҚжңәеҲ¶зҡ„еҺҹзҗҶжҳҜпјҡеңЁдёҖе®ҡзҡ„ж—¶й—ҙй—ҙйҡ”еҗҺпјҢжЈҖжөӢ key зҡ„еҸҳеҢ–жғ…еҶөпјҢз„¶еҗҺжҢҒд№…еҢ–ж•°жҚ®гҖӮ

еңЁredis.windows.confж–Үд»¶дёӯпјҢжңүеҰӮдёӢеҮ иЎҢж•°жҚ®пјҡ

еҰӮжһңжғіиҰҒиҮӘе®ҡд№үж—¶й—ҙе’Ң key зҡ„ж•°йҮҸпјҢзӣҙжҺҘдҝ®ж”№й…ҚзҪ®ж–Үд»¶еҚіеҸҜгҖӮ

дҝ®ж”№й…ҚзҪ®ж–Үд»¶еҗҺпјҢйңҖйҮҚеҗҜRedisжңҚеҠЎеҷЁгҖӮжӯӨж—¶пјҢеҗҜеҠЁж—¶йңҖиҰҒжҢҮе®ҡй…ҚзҪ®ж–Үд»¶еҗҚз§°гҖӮ

еңЁжңҚеҠЎеҷЁзӣ®еҪ•дёӢиҝӣе…Ҙ CMD е‘Ҫд»ӨиЎҢзӘ—еҸЈз•ҢйқўпјҢиҫ“е…Ҙд»ҘдёӢе‘Ҫд»Өпјҡ

2 Jedis ж“ҚдҪң Redis еёёз”Ёзҡ„ API

еңЁдёҠйқўзҡ„demoдёӯпјҢеҲӣе»әе®ўжҲ·з«ҜиҝһжҺҘеҜ№иұЎJdeis е’Ң е…ій—ӯJdeisе®ўжҲ·з«ҜдёҺRedisжңҚеҠЎеҷЁд№Ӣй—ҙзҡ„иҝһжҺҘ жҳҜдёӨдёӘеӣәе®ҡж“ҚдҪңпјҢзңҹжӯЈж“ҚдҪң Redis ж•°жҚ®еә“зҡ„жӯҘйӘӨжҳҜз¬¬2жӯҘзҡ„е…·дҪ“ж“ҚдҪңпјҢжҲ‘д»¬еёёз”Ёзҡ„ API еҰӮдёӢпјҡ

йҰ–е…ҲпјҢжҲ‘д»¬е°ҶиҺ·еҸ– Jedis е®ўжҲ·з«ҜеҜ№иұЎ е’Ң йҮҠж”ҫиҝһжҺҘ зҡ„ж“ҚдҪңжҸҗеҸ–еҮәжқҘпјҒ

2.1 еӯ—з¬ҰдёІзұ»еһӢпјҲStringпјү

  1. еӯҳеӮЁпјҡset key value
  2. иҺ·еҸ–пјҡget key
  3. еҲ йҷӨпјҡdel key

2.2 е“ҲеёҢзұ»еһӢпјҲHashпјү

  1. еӯҳеӮЁпјҡhset key field value

  2. иҺ·еҸ–пјҡ

    • иҺ·еҸ–жҢҮе®ҡfieldеҜ№еә”зҡ„еҖјпјҡhget key field

    • иҺ·еҸ–жүҖжңүзҡ„filedе’Ңvalueпјҡhgetall key

  3. еҲ йҷӨпјҡhdel key field

2.3 еҲ—иЎЁзұ»еһӢпјҲListпјү

  1. ж·»еҠ пјҡеҸҜд»Ҙж·»еҠ дёҖдёӘе…ғзҙ еҲ°еҲ—иЎЁзҡ„еӨҙйғЁпјҲе·Ұиҫ№пјүжҲ–е°ҫйғЁпјҲеҸіиҫ№пјү

    • lpush key valueпјҡе°Ҷе…ғзҙ еҠ е…ҘеҲ—иЎЁе·Ұиҫ№
    • rpush key valueпјҡе°Ҷе…ғзҙ еҠ е…ҘеҲ—иЎЁеҸіиҫ№

  2. иҺ·еҸ–пјҡlrange key start endпјҢж №жҚ®иҢғеӣҙиҺ·еҸ–гҖӮ0 -1иЎЁзӨәиҺ·еҸ–е…ЁйғЁе…ғзҙ гҖӮ

  3. еҲ йҷӨпјҡ
